# Movie Ticket Booking System - SQL Project

## Overview

This project is a Movie Ticket Booking System developed using MySQL. It manages users, movies, theaters, show timings, seat bookings, and payment details.

## Features

* User Registration
* Movie Management
* Theater Management
* Show Scheduling
* Seat Booking
* Payment Tracking
* Booking History

## Database Tables

* Users
* Movies
* Theaters
* Shows
* Seats
* Bookings
* Payments

## Technologies Used

* MySQL
* SQL Queries
* Joins
* Constraints
* Aggregate Functions

## How to Run

1. Create a database in MySQL.
2. Open the `movie_ticket_booking.sql` file.
3. Execute the queries.
4. Run the sample SELECT queries to view the results.
